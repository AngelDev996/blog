# Mi Blog Personal
Plantilla para la realización de un blog mediante el uso del Framework [Astro] (https://astro.build/)
 
 
## Deployment
 
### Para instalar dependencias:
```
npm i
```
 
 
### Para hacer un Build listo a producción:
```
npm build
```
 
 
### Para hacer despliegue en el servidor:
```
npm start
```
